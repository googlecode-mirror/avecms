Сборка из SVN не инсталится!!!
Утомительно после каждой правки готовить инсталяционный пакет.
Поэтому я выкладываю дампы БД для развертывания сборки из SVN.

Не надо извлекать/экспортировать весь репозитарий.
Вам нужна только http://svn.1gb.ru/avecms/trunk
Доступ к репозитарию:
Логин:	public
Пароль:	public

В корне сборки лежит dump.sql который надо импортировать в Вашу БД.
После этого надо в файле inc/db.config.php указать параметры соединения с БД.
Пример файла db.config.php
<?php
$config['dbhost'] = "localhost";	// адрес MySql сервера
$config['dbuser'] = "root";			// имя пользователя БД
$config['dbpass'] = "";				// пароль пользователя БД
$config['dbname'] = "ave209";		// название базы в которую импортирован дамп
$config['dbpref'] = "cp";			// префикс таблиц AVE (изменять не надо)
?>

Вот и весь процесс развертывания сборки из SVN

Желательно почитать комментарии в конфигурационном файле inc/config.php
